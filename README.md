## .gitignore для Terraform

`terraform/.gitignore`:

- Каталог `.terraform/` — содержит локально загруженные провайдеры и модули.
- Файлы состояния `*.tfstate` и их резервные копии — они содержат чувствительную информацию о ресурсах.
- Файлы `*.tfvars` — часто используются для хранения секретов (паролей, ключей).
- Логи сбоев (`crash.log`), временные файлы (`*~`, `.DS_Store`) и служебные файлы IDE.
